# Link Validation

- Keep the reader button text exactly: `Open original`.
- Every source URL must be real and tested.
- If a URL contains a fragment, verify that the exact element id exists and lands near the intended story.
- Do not invent anchors.
- If no per-story anchor exists, use the cleanest stable real page.
- Broken, generic, or misleading links must be fixed or the story must be rejected.
- Keep reader-facing source lines elegant and human, such as `Source: The Dawn-Breakers`, `Source: Memorials of the Faithful`, `Source: Bahá'í Reference Library`, or `Source: Star of the West`.
- Never expose backend phrasing like verified markers, stable anchors, generic landing page, nearest source page, scraping, or automation.
