# Verbatim Text Rules

Every story body shown on Clara's Stories must be a full standalone story or complete source entry copied verbatim from the cited source text.

- Do not paraphrase, summarize, modernize, compress, regenerate, stitch excerpts, or write teaser prose.
- Editorial framing is allowed only in title, summary, tags, read-time label, quote snippet, and source metadata.
- The `story` array itself must contain only source text.
- Preserve source wording and paragraph order except for minor cleanup of OCR/web artifacts, HTML entity decoding, and removal of invisible anchors or duplicated navigation cruft.
- For official Reference Library memorials or chapters, copy the entire memorial/chapter section, not selected paragraphs.
- For books and long works, scan multiple sections first. Choose a passage only if it is complete, self-contained, and does not end abruptly or depend heavily on missing context.
- Never default to the first chapter or easiest passage.
